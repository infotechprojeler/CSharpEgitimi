CSS Giriş Eğitimi

Css Nedir?

CSS: Cascading style sheet in kısaltmasıdır. Ekrandaki html öğelerini tasarlamamızı sağlar.

Css Yazma Seçenekleri

  1. Inline : Direk html elementlerinin içerisine css özellikleri tanımlayabiliriz.
  2. Sayfada head tagları arasına style tagları açarak css yazabiliriz
  3. Harici bir dosya oluşturup (stil.css gibi) sayfalarımızda bu dosyayı çağırarak

Inline Css Yazımı

Elementlerin başlangıç taglarına boşluk bırakıp style attribute ü içine yazabiliriz
Eğer birden fazla css özelliği ekleyeceksek aralarına ; koymalıyız.

Head Kullanarak Css Yazımı

Web sayfalarının en üstünde yer alan head taglarına yazılır.

Harici Css Dosyası Kullanımı

Web projelerinde genellikle harici bir css dosyası oluşturulur ve tüm proje bu dosya üzerinden tasarlanır.
Projeye sağ tıklayıp açılan menüden Add > Style Sheet e tıklayıp css dosyasına bir isim verip ok.

Projeye eklediğimiz css dosyasını sayfamızın head tagları arasında çağırmazsak dosya sayfaya etki etmez!!

CSS Seçiciler

1-Element Seçici

Sayfadaki html elementlerini tag larına göre(body, div, p, h1,2,3..) seçerek css uygulayabiliriz.

2-Id Seçici

Sayfadaki elementleri id özelliği tanımlayarak seçebiliriz.
Id seçici genellikle 1 sayfada 1 element seçmek için kullanılır.
Css dosyasından # işareti kullanılarak sayfadaki id değerine ulaşılır! Örneğin: #idSecici{ buraya seçtiğimiz element için yazacağımız css kodları gelir }

3-Class(Sınıf) Seçici

Sayfalarımızda veya uygulama genelinde birden fazla elemente aynı css özelliklerini uygulamak istersek sınıf seçici kullanırız.

Elementlere class özelliği ekleyip istediğimiz isimle css class ı oluştururuz.

Bir elemente 1 den fazla class tanımlanabilir. Aralarına boşluk bırakarak kullanılır.

Anadiv içindeki ilk p

Bir elementin içindeki elementleri hedefleyerek de css uygulayabiliriz.

Bunun için ana elementi seçip boşluk veya > işareti ekleyerek seçim yaparız.

Anadiv içindeki son p

1 Elemente 1 den fazla css eklenirse

  1. Satır içi stil(elemente inline css yazımı) geçerli olur
  2. Dış css dosyası veya sayfa içindeki style alanındaki css ler
  3. Tarayıcının varsayılan css kodları

Css ile Elementlere Kenarlık (Border) Uygulama

Border solid : düz çizgi

Border dashed : kesik çizgi

Border dotted : noktalı çizgi

Border double : çift çizgi

Border groove : groove çizgi
Border inset : içeri doğru çizgi
Border outset : dışarı doğru çizgi
