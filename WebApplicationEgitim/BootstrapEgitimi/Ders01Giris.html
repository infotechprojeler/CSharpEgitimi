Bootstrap Eğitimi

Bootstrap twitter yazılımcıları tarafından geliştirilen, responsive (her ekran için uyumlu tasarıma sahip) web uygulamaları geliştirmemizi sağlayan, içerisinde hazır css ve javascript kütüphanesi olan bir pakettir.

Bootstrapi Projeye Yükleme

  1. Nuget paket yöneticisi üzerinden
  2. Bootstrap sitesinden indirip manuel yükleyebiliriz
  3. Hazır cdn kütüphane sitelerinden alacağımız kodu sayfaya ekleyerek kullanabiliriz

Bootstrap ekranı 12 eşit parçaya bölerek kullanır. Exceldeki grid-ızgara mantığıyla elemenaları konumlandırır.

Containers - Kapsayıcı

container sınıfı kullanarak içeriği ortalayabileceğimiz kapsayıcı bir element kullanabiliriz.
container-fluid sınıfı kullanarak içeriği tüm ekranı kapsayan bir element kullanabiliriz.
